# ツCKヤ DPS Meter

A real-time DPS meter for **Throne and Liberty** that parses combat logs and displays detailed analytics via a web dashboard.

[![Download](https://img.shields.io/badge/Download-Latest-22d3ee?style=for-the-badge)](https://github.com/mjb6967/CKdpsApp/releases/latest)

## Features

### 📊 Build Testing
- **60-Second Standard Tests** — Industry-standard test duration for fair comparisons
- **Real-Time DPS Tracking** — Live updates as you fight
- **Crit/Heavy/Normal Rates** — With adjustable skill exclusions
- **Per-Skill Breakdown** — Damage, hits, crits, heavies per skill
- **Weapon Assignment** — Assign skills to weapons for weapon-specific stats

### 🎯 Rotation Analysis
- **DPS Timeline** — Visualize damage over time with interactive charts
- **Piano Roll** — See exactly when each skill hit during the fight
- **Gap Detection** — Identify downtime in your rotation
- **Segment Analysis** — Compare quarters (0-15s, 15-30s, 30-45s, 45-60s)

### 💾 Build Management
- **Save Encounters** — Tag with build names and class for organization
- **Class Grouping** — Saved encounters organized by class
- **Load Encounters** — Review saved encounters in full detail
- **Notes** — Add notes to each saved encounter

### ⚔️ Build Comparison
- **Compare Up to 3 Builds** — Side-by-side analysis
- **Skill Breakdown** — See which skills perform better per build
- **Skill Timeline** — Compare rotation timing visually
- **Rotation Analysis** — Segment-by-segment DPS comparison
- **Winner Indicators** — Clear highlighting of best performers

### 🏰 Dungeon Run Builder
- **Track Full Runs** — Combine multiple encounters into a dungeon run
- **Boss Detection** — Automatically identifies boss encounters
- **Run Summaries** — Overall stats, boss breakdowns, encounter timeline
- **Custom Dungeons** — Add your own dungeon types and tiers

### 👥 Party DPS (Beta)
- Share your DPS with party members in real-time
- View party-wide damage rankings
- Requires Discord authentication

## Requirements

- Windows 10/11
- Throne and Liberty with Combat Logging enabled
- Any modern web browser (Chrome, Firefox, Edge)

## Installation

1. Download `TL-DPS-Meter.zip` from [Releases](https://github.com/mjb6967/CKdpsApp/releases/latest)
2. Extract to any folder
3. Run `TL-DPS-Meter.exe`
4. Your browser opens automatically

**No installation required** — fully portable, run from anywhere.

## Enable Combat Logging (Required)

Combat logging must be enabled in-game:

1. Open **Settings** → **Shortcuts** → **Ring Menu Settings**
2. Add **"Combat Meter"** to your Ring Menu
3. In-game, open Ring Menu and activate **Combat Meter**
4. A small combat log icon appears on screen when active

Logs are saved to: `%LOCALAPPDATA%\TL\Saved\CombatLogs`

## Usage

### Basic Workflow
1. **Enable combat logging** in-game
2. **Run** `TL-DPS-Meter.exe`
3. Go to **Build Testing** tab
4. **Reset Encounter** before each pull
5. Fight for 60 seconds
6. **Save** with a build tag and class

### Global Hotkey
| Hotkey | Action |
|--------|--------|
| `Ctrl+Tab` | Reset Encounter (works while in-game) |

### Tabs Overview

| Tab | Description |
|-----|-------------|
| **Dashboard** | Quick overview and navigation |
| **Build Testing** | Main testing interface with Summary, Skills, Weapons, Rotation |
| **Run Builder** | Create dungeon runs from encounters |
| **Run Summary** | View saved dungeon runs |
| **Saved** | List of saved encounters grouped by class |
| **Compare** | Side-by-side build comparison |
| **Skill Settings** | Mark skills that can't crit/heavy |
| **Log File** | Raw combat log (persistent across resets) |
| **Party DPS** | Share stats with party (beta) |

## Configuration

Click **⚙️ Settings** to configure:

- **Log Path** — Auto-detected, or set manually
- **Player Name** — Filter to only show your damage
- **Hotkey** — Change the global reset hotkey (default: Ctrl+Tab)
- **Sound** — Toggle hotkey confirmation beep

All data stored locally in the app folder:

| File | Purpose |

| `config.json` | Settings (log path, player name, hotkey) |
| `encounters.json` | Saved encounters with full stats |
| `runs.json` | Saved dungeon runs |

## Safety & Privacy

**What it does:**
- Reads combat log CSV files generated by the game
- Runs entirely on your local machine
- Stores data locally in JSON files

**What it does NOT do:**
- Inject into the game process
- Modify any game files
- Send data anywhere (except opt-in Party DPS)
- Access any files outside the combat log folder
- Require admin privileges

## FAQ

**Q: Why don't I see damage during combat?**  
A: Throne and Liberty writes combat logs when you leave combat, not during. This is game behavior, not the meter.

**Q: Stats seem delayed?**  
A: The game buffers log writes. Wait until combat ends to see complete data.

**Q: Can I get banned?**  
A: This tool only reads log files the game creates. It doesn't inject, modify, or interact with the game process. Use at your own discretion.

**Q: How do I filter to only my damage?**  
A: Enter your character name in Settings → Player Name.

**Q: Antivirus flags the exe?**  
A: False positive from PyInstaller packaging. The source is available for review.
